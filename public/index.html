Variáveis globais

Valem para todos os hosts e execuções, a menos que sejam sobrescritas.

Perfis (segmentos)

Um perfil agrupa valores de variáveis para um contexto (cliente, ambiente…). Escolha o perfil na hora de executar.

Precedência (a maior vence): sobrescritas da execução > variáveis do host > perfil > globais. Variáveis embutidas: {{host.name}}, {{host.host}}, {{host.port}}, {{host.user}}.

Ranges e listas: uma variável pode guardar uma faixa, ex.: VLANS=100-110,150,200-205. Use literal com {{VLANS}} ou expanda com @cada VLAN em {{VLANS}}: comando com {{VLAN}} no playbook (um comando por item).

Tema

Escolha entre o modo claro (padrão) e escuro. A preferência é lembrada nesta máquina. O terminal e blocos de código continuam escuros nos dois temas.

Inteligência Artificial

Os recursos de IA (assistente e agente no terminal, e geração de playbooks) usam a API da Anthropic (Claude). Informe sua própria chave — ela fica salva só nesta máquina (data.json, permissão 600) e nunca é enviada ao navegador. Precisa de créditos ativos na sua conta Anthropic.

Sugestões: claude-opus-4-8 (mais capaz), claude-sonnet-5 (equilíbrio), claude-haiku-4-5 (rápido/barato).

Aparência do terminal

A fonte e o tamanho valem para o terminal interativo. Veja a prévia abaixo e clique em Salvar para aplicar.

demo@servidor:~$ echo "Olá {{host.name}}" 0123456789 |>_

Dica: fontes como Fira Code, JetBrains Mono ou Cascadia Code só aparecem se estiverem instaladas no sistema; caso contrário o terminal usa uma fonte monoespaçada padrão.

Backup da configuração

Exporta ou importa toda a configuração — playbooks, hosts, perfis e variáveis globais — em um arquivo XML. Ao importar, itens com o mesmo nome são atualizados e nada é apagado.
